Site Name: Nairobi Running Clubs

This name directly represents the website’s focus: running clubs in Nairobi. It is clear, concise, and easy to remember.

Optional domain availability: nairobirunningclubs.com

Site Purpose

The website provides a centralized resource for runners in Nairobi to find and connect with running clubs.

It offers a beginner’s guide for new runners, a directory of clubs with key details, and a contact page for inquiries and updates.

The goal is to foster a strong running community by promoting inclusivity, health, and fitness.

Scenarios

  • "How do I choose the right running club in Nairobi as a beginner?"
  • "Where can I find a list of running clubs with their meeting schedules and locations?"

Color Schema

  • Primary Color: Green (#3B7D32) – Used for headers and key buttons, symbolizing health, growth, and nature.
  • Secondary Color: Orange (#F7941D) – Used for accents like links, highlights, and call-to-action buttons, representing energy and enthusiasm.
  • Background Color: Light Gray (#F5F5F5) – Used for overall background to maintain readability and a clean layout.

Typography

  • Heading Font: Montserrat – Used for headings to ensure modern, bold, and readable titles.
  • Body Font: Open Sans – Used for general text to enhance readability and user experience.
  • Accent Font: Roboto – Used for call-to-action buttons and special highlights.

Wireframe

Mobile View:

  • Navbar with a simple menu (Home, Running Clubs, Contact Us).
  • Hero section with a welcome message and call-to-action.
  • Quick links to club directory and beginner’s guide.
  • Footer with social media links and contact details.

Desktop View:

  • Header with logo and navigation menu.
  • Main section featuring an introduction, running club highlights, and featured articles.
  • Running clubs directory displayed in a grid/list format with filters.
  • Contact form and additional resources in the footer.

CSS

  • Will follow the selected color schema and typography.
  • Responsive design for optimal viewing on mobile and desktop.
  • Clean and minimalistic layout with clear sections and easy navigation.
